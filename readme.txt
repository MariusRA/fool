This app is designed to keep tracking the amount of hours you worked on a project or in several projects on the course of a day, week, month or so.
You should be able to see the amount of hours you worked any day and at any time you want
The toolset used for this application would be for the moment Android Studio version 3.6.3, git and github for task management and repository, and the test device would be for now a Nexus 5 with Android 6.0 Marshmallow
